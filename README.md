# AndroidLibrary
this is toast library

-> Step 1. Add the JitPack repository to your build file

    allprojects {
        repositories {
          ...
          maven { url 'https://jitpack.io' }
        }
      }
      
-> Add the dependency

    dependencies {
              implementation 'com.github.PansuriyaJay:AndroidLibrary:Tag'
      }
